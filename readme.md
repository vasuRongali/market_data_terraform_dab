# Market Data Terraform DAB

Infrastructure as Code (IaC) project for deploying the Market Data platform using Terraform and Databricks Asset Bundles (DAB).

## Project Structure

```
.
├── modules/
├── environments/
├── terraform/
├── databricks/
└── README.md
```

## Prerequisites

- Terraform
- Databricks CLI
- Azure/AWS credentials
- Git

## Getting Started

```bash
git clone https://github.com/<your-username>/market_data_terraform_dab.git
cd market_data_terraform_dab
```
